# ⚔️ RPG Game

A 2D action RPG prototype built with Unity. Protect the girl in the center of the arena from waves of skeletons attacking from both sides.

---

## 🎮 Gameplay

- A girl stands in the center of the arena — protect her at all costs
- Skeletons randomly spawn from the left and right and march toward her
- Slash them with your sword before they reach her
- If a skeleton reaches the girl, it's game over

---

## 🕹️ Controls

| Input | Action |
|---|---|
| `A` | Move left |
| `D` | Move right |
| `Space` | Jump |
| `Left Click` | Attack (sword slash) |

---

## ✨ Features

- 🛡️ **Protect mechanic** — guard an NPC from incoming enemies
- 💀 **Random skeleton spawning** — enemies appear from either side unpredictably
- ⚔️ **Melee combat** — slash enemies with your sword before they reach the target
- 🎮 **Character animations** — idle, movement, jump, and fall states

---

## 🛠️ Built With

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/TarikIsin/rpg-game.git
   ```
2. Open the project in **Unity 2022.3+**
3. Open the scene under `Assets/Scenes/`
4. Press **Play**
